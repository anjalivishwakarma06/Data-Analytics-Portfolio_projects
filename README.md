## 📊 Project 2: Corporate ETL Data Cleaning Pipeline & Report
*An end-to-end data integration project focused on standardizing fragmented legacy database tables using Power Query.*

### 🛠️ Core Engineering Steps
- **Data Integration:** Automated a query append workflow to cleanly stack split transaction logs vertically.
- **Statistical Imputation:** Handled structural null records using data-driven median imputation to safeguard metric distribution profiles.
- **Schema Validation:** Enforced casing guidelines, dropped duplicated rows from market logs, and converted numeric serial data stamps back to calendar elements.
